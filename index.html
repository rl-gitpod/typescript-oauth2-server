# TypeScript OAuth2.0 Server

(opens new window) (opens new window) (opens new window) (opens new window) (opens new window)

@jmondi/oauth2-server is a standards compliant implementation of an OAuth 2.0 authorization server for Node, written in TypeScript.

Requires node >= 12

Out of the box it supports the following grants:

  • Authorization code grant (opens new window)
  • Client credentials grant (opens new window)
  • Refresh grant (opens new window)
  • Implicit grant (opens new window) // not recommended
  • Resource owner password credentials grant (opens new window) // not recommended

The following RFCs are implemented:

  • RFC6749 “OAuth 2.0” (opens new window)
  • RFC6750 “ The OAuth 2.0 Authorization Framework: Bearer Token Usage” (opens new window)
  • RFC7519 “JSON Web Token (JWT)” (opens new window)
  • RFC7636 “Proof Key for Code Exchange by OAuth Public Clients” (opens new window)
